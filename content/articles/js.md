**「Claude Code Skill Hub」** 是一个轻量的静态网站，专门收集和分享 **Claude Code** 的 Skills。

在这里，你可以找到让 Claude Code 变成代码审查员、数据工程师、自动化脚本专家的实用技能包，也可以把自己打磨好的“专家指令”分享给所有 Claude Code 用户。

## 什么是 Claude Code Skill？

Claude Code Skill 是运行在终端里的 AI 程序员 Claude Code 的 **可复用指令集**。
它把领域知识、操作流程和工具权限封装在一个 Markdown 文件里，让 AI 从通用助手瞬间变成专精某项任务的可靠搭档。

## 本站特色

**纯静态，超快加载**：无需登录，无数据库，专注展示内容本身。 

**开箱即用**：每个 Skill 都是一个 `.md` 文件，下载后放入项目的 `.claude/skills/` 目录，一行命令即可激活。

**聚焦 Claude Code**：所有 Skills 均基于 Claude Code 的原生 Skill 系统设计和测试，确保完美兼容。
